# 本文基于tensorflow、keras/pytorch实现对自然场景的文字检测及端到端的OCR中文文字识别

# 实现功能

- [ ]  文字方向检测 0、90、180、270度检测 (运用vgg16分类模型实现，正在整理中)
- [x] 文字检测
- [x] 不定长OCR识别 

## 环境部署
``` Bash
sh setup.sh
```
